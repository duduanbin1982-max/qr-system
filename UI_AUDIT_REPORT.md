## 二、🔴 P0 问题 — 颜色体系断裂

### 2.1 硬编码主色未使用 CSS 变量

CSS 已定义 `--primary: #4F46E5`、`--primary-dark: #4338CA`，但组件中**直接写死**：

| 颜色 | 出现次数 | 应替换为 | 
|------|----------|----------|
| `#4F46E5` | 126 处 (模板72 + JS 54) | `var(--primary)` 或 CSS 类 |
| `#4338CA` | 29 处 (模板15 + JS 14) | `var(--primary-dark)` |
| `#7C3AED` | 11 处 (模板6 + JS 5) | 新增 `--primary-accent` |
| `#059669` | 31 处 | `var(--success)` |
| `#DC2626` | 26 处 | `var(--danger)` |
| `#D97706` | 15 处 | `var(--warning)` |
| `#2563EB` | 4 处 | `var(--info)` |

**影响**: 主色改为 `#6366F1` 只需改 `:root` 一行，但 242 处硬编码全部不会变 → 产生视觉分裂。

### 2.2 旧灰色未替换

CSS v2 已将灰色从 `#9CA3AF` 提升到 `#6B7280` (WCAG AAA)，但组件模板中：

| 旧颜色 | 出现次数 | 新颜色 | WCAG |
|--------|----------|--------|------|
| `#9CA3AF` | 265 处 (模板146 + JS 119) | `var(--text-placeholder)` #6B7280 | 4.5:1 |
| `#94a3b8` | 25 处 | `var(--text-muted)` #4B5563 | 6.8:1 |
| `#999` | 9 处 | `var(--text-muted)` | |
| `#6b7280` (小写) | 23 处 | `var(--text-muted)` | |

**影响**: 灰色文字在浅色背景上对比度不足，可读性差。`#9CA3AF` 在 `#FFFFFF` 上仅 **2.94:1**（WCAG AA 要 4.5:1）。

## 三、🟡 P1 问题 — 间距体系未对齐
